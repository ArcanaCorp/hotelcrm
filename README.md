# Ándale Hotel CRM

Sistema moderno de gestión hotelera desarrollado por ARCANA CORP para hoteles, hostales y alojamientos turísticos que buscan digitalizar y optimizar sus operaciones de manera rápida, profesional y escalable.

Ándale Hotel CRM está diseñado especialmente para mercados turísticos y hoteleros en crecimiento, ofreciendo una experiencia moderna, intuitiva y enfocada en la operación diaria del negocio.

---

# Descripción

Ándale Hotel CRM es una plataforma de administración hotelera que permite gestionar procesos esenciales desde una única interfaz centralizada.

El sistema está orientado a mejorar:
- organización operativa
- control de reservas
- administración de habitaciones
- gestión de huéspedes
- monitoreo de ocupación
- seguimiento de ingresos
- productividad del personal

La plataforma combina una experiencia visual moderna con herramientas prácticas para facilitar la operación diaria de hoteles y alojamientos turísticos.

---

# Módulos Principales

## Dashboard
Panel principal con métricas y estadísticas en tiempo real:
- ocupación hotelera
- reservas recientes
- ingresos
- clientes nuevos
- actividad operativa

---

## Reservas
Gestión completa de reservas:
- check-in
- check-out
- estados de reserva
- historial
- seguimiento de pagos
- observaciones

---

## Habitaciones
Administración de habitaciones:
- disponibilidad
- estados
- precios
- capacidad
- fotografías
- características y servicios

---

## Clientes
Gestión de huéspedes:
- datos personales
- DNI
- historial de estadías
- información de contacto
- observaciones internas

---

## Configuración
Configuración general del sistema:
- información del hotel
- branding
- usuarios
- preferencias operativas
